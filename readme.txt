== Description ==

A MailPro API interface to add email addresses into a MailPro address book

== Installation ==

1. Install the YourChoice MailPro plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the `/wp-content/plugins/` directory).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'YourChoice MailPro' settings page to configure it.
4. Use the YourChoice MailPro Widget or place '[yourchoice_mailpro]' in a page to let your visitors subscribe into your MailPro address book. That's it.
